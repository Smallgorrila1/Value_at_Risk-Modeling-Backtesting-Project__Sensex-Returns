# Value-at-Risk-VaR-Modeling-Backtesting-Project-Sensex-Returns
Implemented a complete multi-model VaR framework using actual Sensex market data. Built Historical, EWMA, Parametric, and EVT-POT VaR models; conducted EVT threshold diagnostics; and performed out-of-sample backtesting using LR_uc, LR_ind, and LR_cc tests.
